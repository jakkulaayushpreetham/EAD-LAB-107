Introduction to Mern Stack 

MERN Stack is a JavaScript-based full-stack web development framework that combines MongoDB, 
Express.js, React, and Node.js to build modern web applications. It allows developers to use JavaScript 
for both the client-side (frontend) and server-side(backend) development, making the development 
process faster and more efficient.

Technologies in MERN Stack :

  1. MongoDB
  2. Express.js
  3. React.js
  4. Node.js
